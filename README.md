## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Muna-Yusuf/LeetCode.git
   ```
   
2. Navigate to the exercise you want to review:
   ```bash
   cd LeetCode/exercise-name
   ```

3. Run the solution and test it yourself!
